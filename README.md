A robust REST API for managing product images, built with Node.js, Express, and MySQL.

## 🚀 Features

- 📤 Upload product images
- 📋 Create product entries with associated images
- 📖 Retrieve product information
- 🔄 Update product details
- 🗑️ Delete products
